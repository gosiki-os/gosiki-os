# Security Policy

If you discover a security vulnerability, please contact us at gosiki.org@gmail.com
We will respond within 24 hours.

## Reporting a Vulnerability

If you discover a security vulnerability in Gosiki OS, please report it by sending an email to:

**gosiki.org@gmail.com**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 24 hours and work with you to address the issue.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

Thank you for helping keep Gosiki OS secure!
